# AutomatedPatellarCoordinateSystem
# 髌骨3D模型特征分析代码

本代码用于髌骨3D模型的几何特征提取与分析，支持STL文件读取、曲率计算、平面拟合、球面均匀采样等功能，适用于骨科生物力学的学术研究。


## 核心功能
1. STL模型读取与可视化
2. 曲率分析（最大/最小主曲率、高斯曲率、平均曲率）
3. 髌骨前面平面拟合（法向量计算）
4. 斐波那契球面均匀采样
5. 几何特征提取（长度、宽度、厚度）


## 依赖库
- VTK 9.0+：用于3D模型的读取与可视化（安装参考[VTK官网](https://vtk.org/documentation/)）
- C++11及以上编译器


## 使用方法
1. 编译：通过CMake配置项目，链接VTK库，生成可执行文件。
2. 运行：
   ```bash
   ./your_executable_name  # 按提示输入STL文件名和参数
3. 输入：STL模型文件（需用户自行准备）。
4. 输出：
   拟合平面法向量、球面采样点等特征数据（CSV格式）
   归一化后的STL模型（可选）
